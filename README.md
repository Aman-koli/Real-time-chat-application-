# Real-Time Chat Application

A full-stack real-time chat application built with React.js, Django, and WebSockets, enabling users to exchange messages instantly with a modern and responsive user interface.

## 🚀 Tech Stack

- Frontend: React.js
- Backend: Django
- Real-Time Communication: WebSocket
- Database: SQLite (or PostgreSQL/MySQL if you're using one)

## ✨ Features

- Real-time messaging
- User authentication
- Responsive UI
- Fast WebSocket communication
- Clean and modern interface

## 📦 Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```
